## OpenMP Example

Example for using OpenMP

## Installation

### Setup OSX

OpenMP header files are not included in Xcode gcc/g++, thus needs to perform
the following:

```
    brew install gcc
```

Then set globals:

```
    export CC=/usr/local/bin/gcc-8
    export CXX=/usr/local/bin/g++-8
```
